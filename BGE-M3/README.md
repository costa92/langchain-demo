# 使用BGE-M3和K近邻实现语义搜索

## 环境安装

```sh
# 安装BGE-M3
pip install -U FlagEmbedding -i https://pypi.tuna.tsinghua.edu.cn/simple

# 安装scikit-learn
pip install scikit-learn -i https://pypi.tuna.tsinghua.edu.cn/simple
```
